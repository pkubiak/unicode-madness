# unicode-madness

Simple clone of Marble Maddness / Neverball created during [gamejam](https://github.com/pkubiak/gamejam). 

**Objective**: Collect all unicodes in correct order, avoid void!

## Controls ##
- desktop: Mouse
- mobile: smartphone rotation (via motion sensor)
